## ğŸ“‹ Fitur Utama
Sesuai dengan studi kasus manajemen keuangan, aplikasi ini mencakup:
1.  **Sistem Autentikasi**: Registrasi/Daftar dan Login akun yang aman.
2.  **Dashboard Ringkasan**: Menampilkan total saldo, total pemasukan, dan total pengeluaran secara real-time.
3.  **Manajemen Transaksi (CRUD)**:
    * Mencatat Pemasukan.
    * Mencatat Pengeluaran.
    * Mengubah data transaksi.
    * Menghapus data transaksi.
4.  **Riwayat Keuangan**: Daftar detail transaksi yang telah dilakukan oleh pengguna.

## âš™ï¸ Cara Instalasi Lokal
Jika ingin menjalankan proyek ini di komputer lokal:

1.  **Clone Repository:**
    ```bash
    git clone [https://github.com/deynaangelyzahara/UAS-SI-VB.git](https://github.com/deynaangelyzahara/UAS-SI-VB.git)
    ```

    * Jalankan server: `npm start`.

3.  **Konfigurasi Frontend:**
    * Masuk ke folder `cd frontend-react`.
    * Install library: `npm install`.
    * Jalankan aplikasi: `npm run dev`.
